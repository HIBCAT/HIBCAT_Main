HIBCAT

HIBCAT is the prototype project which aims to perform ML/AI based R and Python functions of real time data collected by it's users. This site is intended to serve as the data dictionary of all the tables in all the databases from all the sourced grids.



For the current project, please visit 'Data Grid 1' and 'Data Grid 2' options from the menu. 'HIBCAT' menu will not contain any links at the time of submissions.
